# InventoryMaster Frontend 🎨

The visual heart of the Inventory Management System. A high-performance, responsive React application built with precision and aesthetics.

## ✨ Highlights

- **⚡ Blazing Fast**: Built with Vite for near-instant HMR and optimized production builds.
- **💎 Modern Aesthetics**: Beautifully crafted with Tailwind CSS v4, featuring glassmorphism and smooth transitions.
- **📊 Interactive Data**: Dynamic data visualization using Recharts for stock analytics.
- **🎭 Smooth Animations**: Powered by Framer Motion for a premium desktop and mobile experience.
- **✅ Type-Safe Forms**: Robust form management with React Hook Form and Yup validation.
- **📱 Fully Responsive**: Optimized for all screen sizes, from mobile phones to high-res monitors.

---

## 🛠️ Technology Stack

- **React 19**: Utilizing the latest React features and optimizations.
- **Vite**: Modern frontend tooling for better DX.
- **Tailwind CSS v4**: The cutting edge of utility-first styling.
- **Lucide Icons**: Crisp, consistent iconography.
- **Recharts**: Composited charting library for business intelligence.
- **Framer Motion**: Production-ready motion for UI elements.
- **React Hot Toast**: Beautiful notifications for user feedback.

---

## 🏗️ Architecture

The frontend follows a modern, component-driven architecture:

```bash
📂 src
 ┣ 📂 components   # Shared UI elements (Buttons, Inputs, Modals)
 ┣ 📂 pages        # Page-level containers
 ┣ 📂 context      # Global state (Auth, Theme)
 ┣ 📂 services     # API communication (Axios)
 ┣ 📂 hooks        # Custom business logic
 ┗ 📂 utils        # Helper functions & constants
```

---

## 🚀 Getting Started

1. Navigate to directory: `cd frontend`
2. Install dependencies: `npm install`
3. Configure environment:
   - Create a `.env` file:
     ```env
     VITE_API_BASE_URL=http://localhost:3000/api
     ```
4. Start development server: `npm run dev`
5. Build for production: `npm run build`
